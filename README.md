## About

Black Sea Walnut is a multi-page corporate website for a walnut production and wholesale company. Built with a professional Gulp build system featuring SCSS compilation, Webpack JS bundling, image optimization, SVG sprites and PHPMailer contact form. Includes separate pages for delivery conditions and wholesale corporate clients.

🔗 **Live:** [black-sea-walnut](https://vergos1.github.io/black-sea-walnut/)

## Features

- **PHPMailer** — server-side contact form email delivery
- **Swiper** — touch sliders
- **AOS** — scroll-triggered animations
- **just-validate** — form validation
- **Inputmask** — phone input masking
- **Rellax** — parallax scrolling
- **SVG sprite** — optimized icon system
- **WebP** — automatic image conversion
- **Gulp build** — dev server, minification, cache busting and ZIP export

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Vergos1/black-sea-walnut.git

# Install dependencies
npm install

# Start development server
npm run dev

# Production build
npm run build
```

> **Note:** Contact form requires a PHP server. GitHub Pages serves static files only — deploy to a PHP-capable host (e.g. shared hosting, VPS) for full functionality.
